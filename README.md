# OPML to Miniflux

Import an OPML to Miniflux


## Using

```
opml-to-miniflux [args]

Options:
  --version          Show version number                               [boolean]
  --file, -f         OPML file to import                     [string] [required]
  --url, -u          Miniflux URL                            [string] [required]
  --login, -l        Miniflux login/username                 [string] [required]
  --password, -p     Miniflux password                                [required]
  --category, -c     Miniflux category to import to          [number] [required]
  --concurrency, -n  Concurrency limit                     [number] [default: 5]
  --help             Show help                                         [boolean]
  
```
